# My Food Diary (맛뿌다)

#### 프로젝트 개요

* 사용자 보유한 식재료에 기반해 레시피를 추천해 주는 웹 서비스.



#### 주요 기능

* 사용자가 보유한 식재료를 등록할 수 있다.
  * 식재료 목록을 수정, 삭제할 수 있다.
  * 식재료의 유통기한을 확인할 수 있다.
* 등록한 식재료나 추가로 사용할 식재료를 선택해 레시피를 추천받을 수 있다.
* 추천받은 레시피의 상세 정보를 확인할 수 있다.
